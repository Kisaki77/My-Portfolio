Anixi Health

More about the mobile app

Welcome to more details about the Mobile Anixi Health Application, where users can be comforted and healped with their health issues.
This project was a team work project for a client. I worked on some parts of the application which is front-end and back-end.
We used flutter and dart to develop the application for both Android and iOS devices.
For databased we used Firebase.

I was working on the profile page where the user can view and update thier details.
I was also working on the journal page where the user can view, create, and update thier journal entries.
